Welcome! This is my personal blog where I write about what I have learnt or thought during my daily life. Hope you can enjoy it.

Things I like

I like to learn new things and I enjoy playing with digital devices (smart phones, Kindle, etc.).

I enjoy programming. I am most familiar with Python and fairly proficient in using Numpy, Matplotlib and others. I use Pandoc to generate beautiful document from my daily notes. My favorite editor is Neovim now. I try to set up Neovim to write Python and edit my Markdown notes.

I am a big advocator for Stack Overflow. Thanks to all these guys on Stack Overflow, I really learned a lot, although there are occasionally little bumps. You can find me on Stack Overflow most of the time.

I enjoy reading novels, biography, history etc. One of my long-time favorite is Note of Hiding in the East by Pu Zong, which depicts the lives of academics of China’s top university during the WWII in the remote south western province Yunnan. I like listening to music by Cheer Chen and Deserts Chang.

How is this blog built?

This blog is created using static blog generators and hosted by the wonderful GitHub pages service. I first use Hexo to build this blog. I have switched to Hugo for its high performance. You can find the migration process logged here.
